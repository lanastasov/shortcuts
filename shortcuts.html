Google Chrome Browser
more shortcuts

CTRL+F4 - close currently open tab

CTRL+W - close currently open tab

CTRL+SHIFT+T - restore the last closed tab

CTRL+J - opens chrome://downloads/

CTRL+H - opens chrome://history/

CTRL+U - opens view-source://page-url - a window with source code of the page

CTRL+T - opens new tab

CTRL+D - opens a window to add currently open tab to BOOKMARKS

CTRL+SHIFT+W - exit browser

ALT+D+ENTER - opens the same tab in a new one

ALT+HOME - opens the homepage(chrome://settings/ -> Appearance -> Show Home Button -> Change)
